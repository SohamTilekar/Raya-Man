# Game Overview

## Core Concept
This game is an open-ended, procedurally generated Action RPG set in a Medieval Fantasy world filled with magic, monsters, and sprawling kingdoms. Inspired by popular Isekai anime and manhwa, the game heavily features progression systems (like Ranks and Classes/Tank systems) and dynamic storytelling.

## The AI Story Director (The "Game Master")
The core distinguishing feature of the game is its **AI Story Director**. Powered by a small, lightweight neural network (approx. 5-6M parameters), this system constantly monitors the player's actions, playtime, current Rank, and location. Instead of a linear story, the AI dynamically selects from a pool of "Pre-created Events" and weaves them into the player's journey to create a unique narrative.

## Key Features
1. **Procedural World & Kingdoms:**
   - The world is procedurally generated with diverse biomes, resources, and dungeons.
   - Kingdoms run naturally with simulated economies, politics, and a multitude of NPCs living out their daily lives.
2. **Dynamic Threat & Scaling:**
   - The AI monitors the "Danger Level" around the player.
   - As the player survives, the baseline threat continuously increases, effectively forcing the player to adapt, level up, and increase their Rank to survive.
3. **Isekai/Manhwa Tropes & Systems:**
   - Features classic class roles (Tank, DPS, Support/Healer) combined with a strict Ranking system (e.g., F-Rank to SSS-Rank).
   - "Deus Ex Machina" moments driven by the AI (e.g., being saved by a high-ranking Prince/Princess when near death in the wilderness).
   - **The Returner System:** Roguelite time-regression if defeated by an ultimate Calamity.
   - **Guild Management (Endgame):** Establish guilds, recruit NPCs, and send them on missions to manage the world's danger level.
4. **Living Plot:**
   - No two playthroughs are the same. The plot shifts based on how the AI decides to challenge or reward the player.
